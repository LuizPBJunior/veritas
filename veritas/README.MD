# Veritas

Jogo de perguntas e respostas sobre o catolicismo. Com o objetivo de evangelizar
e trazer mais conhecimentos doutrinhas e históricos da Igreja Católica Aposólica Romana.

Projeto desenvolvido em Python.

Status: Em desenvolvimento.

No primeiro arquivo temos main.py, nele temos a estrutura do código, podemos ver
como o código funciona.

No segundo arquivo temos regras.py, para os jogadores que tiverem interesse em 
conhecer as regras o jogo.

No terceito arquivo temos cartas.py, onde todas as cartas são armazenadas e 
separadas por categoria
